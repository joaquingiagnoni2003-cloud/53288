# Analizador Sintáctico y Semántico - Calculadora ANTLR4

Este proyecto implementa un analizador sintáctico, léxico y un intérprete semántico para una calculadora utilizando ANTLR4 y JavaScript sobre Node.js.

## Requisitos previos
* Node.js (v24 o superior)
* Java SE (para la generación de código ANTLR)
